# Movies

Movie catalog web application developed using ASP.NET Core MVC, Entity Framework Core and SQL Server.

## Live Demo

http://movies-revive.runasp.net/

## Technologies

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* Razor Pages
* Bootstrap

## Features

* View movie catalog
* Search movies by title
* Create new movies
* Edit movie information
* Delete movies
* Pagination support
* Database integration with SQL Server

## Project Structure

* Movies - main web application
* Picture - image management
* TODOList - additional training project
* Blazor - Blazor examples and components
